    Project Locations management workflow¶

    (For testing purposes)

    Welcome to the guide for updating project locations at KfW. This workflow takes project-level location data from collection through validation, upload, aggregation, and map publication on the Open Data Platform (ODP).

    Version: 2026-v0 · Glossary (hover over highlighted terms anywhere for a quick definition)

    New collection or updating existing data?

    • New collection or updating existing collection already on ODP → start at Step 1: Collect
    • Existing collection with older version of PLM → read Updating existing data first

    Steps at a glance¶

    # Step Tool Performer Maintainer
    1 Collect Project Locations Model (PLM) PIA Geodata-Desk
    2 Validate Online validator PIA ZDV (DAI)
    3 Visual plausibility check Online validator KfW (PM/PA) ZDV (DAI)
    4 Upload data ODP KfW (PM/PA) ZDV (DAI)
    5 Aggregate data Python script Geodata-Desk Geodata-Desk
    6 Create map ODP Geodata-Desk ZDV (DAI)
